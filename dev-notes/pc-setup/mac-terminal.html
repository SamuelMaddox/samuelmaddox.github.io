Mac Terminal

Add Brogrammer Color Theme (Mac)​

  1. Download the brogrammer.terminal theme.
  2. Go to Apple Terminal → Settings → Profiles → Circle with 3 dots → Import to import the theme.
  3. Select the theme from the list of themes.
  4. Set the theme as the default profile by clicking on the Default button at the bottom of the settings window.
  5. Update the selection color for the profile to #45A2D2

Install Oh My Zsh​

  1. Install Homebrew if you haven't already: Homebrew Installation
  2. Follow this guide: How to install ZSH on macOS

Case Insensitive Tab Completion​

By default, Zsh "Tab Completion" may be case sensitive. To make it case insensitive when typing lower case letters, open your ~/.zshrc file in an editor and append the following lines:

autoload -Uz compinit && compinit
zstyle ':completion:*' matcher-list 'm:{a-z}={A-Za-z}'
